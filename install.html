1. Choose how to install

2. Download


(c++) Visual Studio 2013 project






Latest Windows Version: 1.1.4.6

Or use the autoupdater in help -> "Check for updates"

Source-Code hosted on Github

3. Prepare LoLUpdater for patching

  • Check if LoL has updates then close the PvP.net client and every other League process
  • Right click the LoLUpdater file -> Properties -> Unblock

    • 32bit OS: MSVC 2013 x86
    64bit OS: MSVC 2013 x64

4. Run LoLUpdater.exe and click the right folder

  • "Standard" League: Riot Games\League of Legends (where lol.launcher.exe is)

  • GarenaLoL: GarenaLoL\GameData\Apps\LoL (where lol.exe is)

  • LoLQQ: 腾讯游戏\英雄联盟 (where lol.launcher_tencent.exe is)

    • Play a custom game first

The LoLUpdater patch will be reset everytime Riot updates League

5. (Optional) Download and Install DirectX Web-Installer

2. Install prerequisites

  • Download Adobe Air Beta

  • Download NvidiaCG Toolkit

    • Yes you will still need this even if you are on ATI hardware

  • Download NPAPI Adobe Flash Beta

  • Close Firefox and your other Netscape compatible browsers

  • Unblock the installers and install them

    • Right click file -> Properties

3. Copy DLL files

Copy all these files to %leaguedirectory%\RADS\solutions\lol_game_client_sln\releases\%Latest%\deploy

Garena/LoLQQ paths: %leaguedirectory%\Game and %leaguedirectory%\Air

  • Copy cg.dll, cgGL.dll and cgD3D9.dll from %WindowsDrive%:\Program Files (x86)\NVIDIA Corporation\Cg\bin

  • 64bit OS: Download msvcp120.dll and msvcr120.dll
  • 32bit OS: Download msvcp120.dll and msvcr120.dll

    • Copy these two files to %leaguedirectory%\RADS\projects\lol_patcher\releases\%Latest%\deploy also

    64bit OS

  • Windows 8/8.1/10 tbb.dll
  • Windows 7 tbb.dll
  • Windows Vista tbb.dll
  • Windows XP tbb.dll

    • (32bit OS) Download CPU-Z and check your SIMD instructions (AVX2 > AVX > SSE2)

  • AVX tbb.dll
  • AVX2 tbb.dll

  • SSE2 tbb.dll
  • AVX tbb.dll
  • AVX2 tbb.dll

4. Copy msvcp110.dll and msvcr110.dll

  • 64bit OS: Download msvcp110.dll and msvcr110.dll
  • 32bit OS: Download msvcp110.dll and msvcr110.dll

    • Copy these two files to %leaguedirectory%\RADS\projects\lol_air_client\releases\%Latest%\deploy

5. Copy Adobe AIR.dll

from %WindowsDrive%:\Program Files (x86)\Common Files\Adobe AIR\Versions\1.0 to %leaguedirectory%\RADS\projects\lol_air_client\releases\%Latest%\deploy\Adobe AIR\Versions\1.0

6. Copy NPSWF32.dll

64bit OS: from %WindowsDrive%:\System32\Macromed\Flash to %leaguedirectory%\RADS\projects\lol_air_client\releases\%Latest%\deploy\Adobe AIR\Versions\1.0\Resources

32bit OS: from %WindowsDrive%:\SysWOW64\Macromed\Flash to %leaguedirectory%\RADS\projects\lol_air_client\releases\%Latest%\deploy\Adobe AIR\Versions\1.0\Resources

Repeat this process after every patch (By Riot), even the micro patches
