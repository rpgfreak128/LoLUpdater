Higher Quality Audio

Go to your speakers-settings and set the "Default Format" to 24bit 192Khz (Might not work on all speaker-setups)


SSD

Boost your Samsung SSD performance (with "Samsung Magician")

Reduce RAM usage in Windows (Reduces prettiness)

Run SystemPropertiesPerformance.exe and select "Adjust for best performance" then click ok to remove the aero theme for Windows (Uses less RAM)

You can then run services.msc and disable the "Themes" service by right clicking -> Properties -> "Startup mode -> Disabled" and then stopping the service, this will disable you to apply any Windows themes though.

Disable startup items

Run msconfig.exe -> Startup and disable all items you do not want at Windows startup

Disable mouse-acceleration

Open Mouse Properties -> Pointer Options and unclick "Enhance pointer precision" -> OK

Free up disk space

Run cleanmgr.exe and select what you want to clean
