Ingame improvements

  • Boost framerate
  • Reduce frame-drops
  • Reduce in-game shop lag

How it Works

Replaces League's DLL files with the latest ones available from Nvidia/Intel/Adobe/Microsoft, (LoL's files are not the newest available).

These are used by League of Legends.exe to run the game, replacing them shows improvement (no bugsplats also).
